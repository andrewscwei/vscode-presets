## Usage

### Installing Presets

Install via cURL:

```sh
$ curl -o- -s https://raw.githubusercontent.com/andrewscwei/vscode-presets/master/install.sh | bash
```

### Exporting Current VSCode presets

```sh
$ git clone https://github.com/andrewscwei/vscode-presets
$ cd vscode-presets
$ ./export.sh
```
